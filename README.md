## 🌟 Overview

**WING** is a social marketplace platform designed specifically for artisans to showcase their "Finished Masterpieces" and "Work in Progress" (WIP) crafts. It bridges the gap between traditional craftsmanship and modern digital commerce, providing a space for discovery, community, and secure transactions.

## ✨ Key Features

- **Masonry Social Feed:** A beautiful, Pinterest-style grid showcasing artisan creations.
- **Dynamic Filtering:** Search by category (Crochet, Woodwork, Ceramics, etc.) or post status (Finished vs. WIP).
- **Secure Buy Flow:** Integrated "Buy Safe" redirection to a verified Telegram Bot for anti-fraud payment handling.
- **Artisan Streak:** Encourages daily creativity by tracking and displaying posting streaks.
- **AI Mentor:** An integrated AI assistant to help artisans with craft techniques and business advice.
- **Multilingual Support:** Full support for English and Amharic languages.
- **Real-time Notifications:** Instant alerts for likes, comments, and community interactions.

## 🛠️ Tech Stack

- **Frontend:** React.js with TypeScript
- **Styling:** Tailwind CSS (v4)
- **Icons:** Lucide-React
- **Backend/Auth:** Firebase (Authentication, Firestore, Storage)
- **Deployment:** Render / GitHub Actions
